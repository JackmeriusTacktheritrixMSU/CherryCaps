**The Official Cherry Brand Cherry Keycap**

My last name is Cherry, and I just made a super awesome keyboard so I wanted to make a cool cherry keycap to put on my new board. I made three keys in total, two centering keys- f and j which look like those letters, and a plain cherry key. 


**Manufacturing**
The idea is that you silicon mold a resin 3D Print, and then you fill the mold with real resin. 

Print the STL models and cover them in silicon. Then fill the silicon mold with resin and let it harden. You want to use green resin to fill the stem and red for the top. If you mess it up a little, you can probably sand a little bit but you'd need to buff it after.
